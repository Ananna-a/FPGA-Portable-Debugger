## 通信链路

```
上位机 → CDC (FX2 USB) → FPGA : 命令帧 55 AA | cmd | len | payload | cs
FPGA  → CH340 UART     → 上位机 : 应答帧 AA 55 | mod_id | func_id | status | data | cs
FPGA  → RGMII UDP      → 上位机 : 数据包 5A AA | seq | flags | 1008B ADC data
```

详见 `03_通信协议.md`。

## FPGA 命令链

```
FX2 USB (48MHz)
  → FX2_CDC_Core (SlaveFIFO, RX状态机: IDLE→READ→WRITE)
  → fifo_in (跨时钟域: 48MHz → 50MHz)
  → cdc_cmd_parser (状态机: IDLE→HEAD2→CMD→LEN_L→LEN_H→PAYLOAD→CS→DONE)
  → debugger_top cmd_valid_flag case语句 (50+ 命令码)
  → 按 cmd_code 区间分发到各子模块
```

## 上位机架构

```
main_app.py 创建全局 SerialManager 实例，传入所有 Tab

SerialManager 是数据总线:
  - 发送: send_command(cmd, payload) → CDC 串口
  - 接收: CH340 串口 → _on_data_received() 状态机分流:
      ① DSA → dsa_measurement_received
      ② 频率 → frequency_data_received
      ③ CAN  → can_data_received
      ④ Bode → bode_data_received
      ⑤ 应答 → log_message

Tab 间通过 SerialManager 的信号间接通信，Tab 本身无直接依赖
```

## 时钟系统

```
外部 50MHz → gowin_pll (ip/)
  ├── 50MHz  → 系统时钟 (debugger_top, 所有逻辑模块)
  ├── 125MHz → DDR3 + 以太网 (scope/ethernet/)
  └── 48MHz  → FX2 USB (comm/FX2_CDC_Core)

ddr_pll (ip/) → DDR3 专用时钟
```

## 共享资源

| 资源 | 使用者 | 说明 |
|------|--------|------|
| `ip/gowin_pll` | 全部 | 系统主时钟源 |
| `ip/ddr3_ctrl_2port` | scope + logic_analyzer | DDR3 双端口，可复用 |
| `ip/fifo_top` | comm + scope + logic_analyzer | 实例化3次 |
| RGMII | scope | 以太网 TX，被 scope 独占 (CAN UDP 通道因缺少仲裁器禁用) |
