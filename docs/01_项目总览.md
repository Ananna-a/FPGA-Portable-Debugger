# 项目总览

多功能协议调试器 — 高云 GW5AT-138B + ACX720 平台。

## 项目概要

- **FPGA 芯片**: GW5AT-138B (GW5AT-LV138PG484AC1/I0)，50MHz 系统时钟
- **开发板**: ACX720（FX2 USB + DDR3 + ADC + DAC）
- **综合工具**: GowinSyn（高云云源软件）
- **工程入口**: `acm2108_ddr3_CDC.gprj`
- **顶层模块**: `debugger_top` (~4200 行)
- **引脚约束**: `src/acm2108_ddr3_CDC.cst` (441 行)

## 架构全景

```
┌─────────────────────────────────────────────────────────┐
│ 上位机 (Python/PySide6)                                  │
│  main_app.py → 5个Tab → SerialManager (单例)             │
│    ├── 示波器 Tab                                        │
│    ├── DDS 函数发生器 Tab                                │
│    ├── 协议转换器 Tab (序列+PWM+设备中心)                   │
│    ├── 逻辑分析仪 Tab                                     │
│    └── 波特图 Tab                                        │
└──────────┬────────────┬──────────────┬──────────────────┘
           │ CDC COM    │ CH340 COM    │ UDP :6102
           ▼ (命令)     ▲ (应答+数据)   ▲ (ADC波形)
┌─────────────────────────────────────────────────────────┐
│ FPGA (debugger_top.v)                                    │
│                                                         │
│  FX2_CDC_Core → fifo → cdc_cmd_parser → 命令路由       │
│    按 cmd_code 区间分发 → 50+ 命令码 → 子模块             │
│                                                         │
│  子模块应答 → uart_response_tx → uart_tx_mux            │
│    → uart_byte_tx → CH340 → 上位机                       │
│                                                         │
│  ADC→DDR3→eth_udp_tx→RGMII→上位机 (示波器数据)           │
└─────────────────────────────────────────────────────────┘
```

## 构建命令

### FPGA 端
打开 `acm2108_ddr3_CDC.gprj` 在高云云源软件中综合/布局布线/生成比特流。

### 上位机端
```powershell
pip install PySide6 numpy pyqtgraph pyserial
cd src\APP
python main_app.py
```
**必须在 `src/APP/` 目录下运行**（相对导入依赖）。

## 通信概览

| 通道 | 方向 | 物理接口 | 速率 | 用途 |
|------|------|---------|------|------|
| CDC | 上位机 → FPGA | FX2 USB | USB 2.0 FS | 命令发送 |
| CH340 | FPGA → 上位机 | CH340 UART | 115200 bps | 应答+数据上报 |
| UDP | FPGA → 上位机 | RGMII 以太网 | 100 Mbps | 示波器ADC波形 |

CDC帧: `55 AA | cmd | len_l | len_h | payload... | cs`  (cs = `~(cmd+len_l+len_h+Σpayload)`)
应答帧: `AA 55 | mod_id | func_id | status | data | cs`

详细命令码表见 `02_通信协议.md`。
