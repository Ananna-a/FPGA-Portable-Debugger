# 通信协议

## 物理层

| 通道 | 方向 | 物理接口 | 波特率/速率 | 用途 |
|------|------|---------|------------|------|
| CDC | 上位机 → FPGA | FX2 USB CDC | USB 2.0 Full Speed | 命令发送 |
| CH340 | FPGA → 上位机 | CH340 USB-UART | 115200 bps | 应答 + 数据上报 |
| 以太网 | FPGA → 上位机 | RGMII UDP | 100 Mbps | 示波器波形数据 |

---

## CDC 命令帧格式

```
字节 0-1:  帧头   0x55 0xAA
字节 2:    cmd    命令码
字节 3:    len_l  载荷长度低字节
字节 4:    len_h  载荷长度高字节 (小端序)
字节 5-N:  payload  载荷数据
字节 N+1:  cs     校验和

CS = ~(cmd + len_l + len_h + Σpayload) & 0xFF
```

## 应答帧格式

```
字节 0-1:  帧头   0xAA 0x55 (与命令帧头相反)
字节 2:    mod_id  模块ID
字节 3:    func_id 功能ID (=命令码)
字节 4:    status  状态 (0x00=成功)
字节 5:    data    附加数据
字节 6:    cs     校验和

CS = (mod_id + func_id + status + data) & 0xFF
```

## 完整命令码表

### 系统控制 (0x00-0x05)

| 命令 | 名称 | Payload | 说明 |
|------|------|---------|------|
| 0x00 | 系统复位 | — | FPGA 全局复位 |
| 0x03 | 设置数码管 | [hex_val(4B)] | 4位数码管显示 |
| 0x04 | 读取状态 | — | 查询 FPGA 状态 |
| 0x05 | 蓝牙使能 | [enable(1B)] | HC-06 蓝牙开关 |

### DDS 函数发生器 (0x10-0x1F)

| 命令 | 名称 | Payload |
|------|------|---------|
| 0x10 | 通道A波形类型 | [wave_type(1B)] |
| 0x11 | 通道B波形类型 | [wave_type(1B)] |
| 0x12 | 通道A频率 | [freq_word(4B, 大端)] |
| 0x13 | 通道B频率 | [freq_word(4B, 大端)] |
| 0x14 | 通道A相位 | [phase(1B)] 0-255 对应 0-360° |
| 0x15 | 通道B相位 | [phase(1B)] |
| 0x16 | 通道A幅度 | [amp(1B)] 0-255 DAC值 |
| 0x17 | 通道B幅度 | [amp(1B)] |
| 0x18 | 通道使能 | [mask(1B)] bit0=A, bit1=B |
| 0x19 | 通道A全部参数 | [type(1)][freq(4)][phase(2)][amp(1)][offset(1)][duty(2)] 大端 |
| 0x1A | 通道B全部参数 | 同上 |
| 0x1C | 通道A占空比 | [duty(2B, 大端)] |
| 0x1D | 通道B占空比 | [duty(2B, 大端)] |
| 0x1E | 通道A任意波形 | [wave_data(256B)] |
| 0x1F | 通道B任意波形 | [wave_data(256B)] |

波型类型: 0=正弦, 1=方波, 2=三角, 3=锯齿, 4=反锯齿, 5=脉冲, 10=任意

### PWM 控制器 (0x50-0x52)

| 命令 | 名称 | Payload |
|------|------|---------|
| 0x50 | PWM配置 | [通道ID(1B)][频率字(4B, 大端)][占空比(2B)] |
| 0x51 | PWM使能 | [掩码(1B)] bit0-7对应CH0-7 |
| 0x52 | PWM停止 | — |

### CAN 总线 (0xC0-0xC4)

| 命令 | 名称 | Payload |
|------|------|---------|
| 0xC0 | 配置波特率 | [baud_index(1B)] |
| 0xC1 | 发送CAN帧 | 标准帧: [type=0][ID_H][ID_L\|DLC][data0-7] |
|      |         | 扩展帧: [type=1][ID3][ID2][ID1][ID0\|DLC][data0-7] |
| 0xC2 | 设置滤波器 | [filter_id(4B)] |
| 0xC3 | 读取状态 | — |
| 0xC4 | 接收数据上报 | (FPGA主动通过 CH340 发送 CAN 帧) |

CAN 接收数据通过 CH340 UART (115200bps) 上报，`SerialManager` 自动识别首字节 0x00/0x01 为 CAN 数据。
